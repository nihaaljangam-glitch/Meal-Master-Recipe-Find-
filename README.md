# Meal Master — Recipe Finder
## Features
Recipe Search — Search for recipes by name, ingredient, or cuisine
Ingredient-Based Lookup — Enter ingredients you have on hand and find matching recipes
Dietary Filters — Filter by vegetarian, vegan, gluten-free, and more
Recipe Details — View full ingredient lists, step-by-step instructions, and nutritional info
Save Favorites — Bookmark recipes to revisit later
## Tech Stack
Frontend: HTML, CSS, JavaScript
API: https://www.themealdb.com/api.php
Storage: LocalStorage for saved favorites
## Usage
Enter an ingredient or recipe name in the search bar
Apply filters for dietary preferences or cuisine type
Click a recipe card to view full details
Click the bookmark icon to save a recipe to your favorites
